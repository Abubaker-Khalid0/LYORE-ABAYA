# LYORE ABAYA

Luxury modest fashion e-commerce website.

## Project Structure

```
├── index.html           # Homepage
├── collections.html     # Product listings
├── product-detail.html  # Single product view
├── contact.html         # Contact & Bespoke orders
├── size-guide.html      # Size guide
├── returns-policy.html  # Shipping & returns
├── assets/
│   ├── js/main.js       # Main JavaScript
│   └── images/          # All images
└── README.md
```

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Getting Started

Simply open `index.html` in a browser to view the website locally.
